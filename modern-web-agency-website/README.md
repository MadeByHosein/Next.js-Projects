# Next.js

[![Next.js](https://img.shields.io/badge/Next.js-16-black)](https://nextjs.org/)
[![Tailwind](https://img.shields.io/badge/Tailwind-4-blue)](https://tailwindcss.com/)

## 🚀 ویژگی‌های پروژه

### 🎨 ظاهر و تجربه کاربری

- **حالت تاریک/روشن خودکار** با استفاده از Theme Provider که با تم سیستم کاربر هماهنگ می‌شود
- **استایل‌دهی حرفه‌ای** با Tailwind CSS
- **کامپوننت‌های قابل استفاده مجدد** (Reusable Components) برای کاهش تکرار کد
- **اسکرول نرم به بالا** با قابلیت Scroll To Top

### ✨ انیمیشن‌ها و جلوه‌های ویژه

#### اسکرول انیمیشن

- استفاده از پکیج **AOS** برای انیمیشن‌های هنگام اسکرول
- افکت‌های fade، slide و zoom با قابلیت تنظیم delay و duration

#### هیرو سکشن (Hero Section)

- **سیستم ذرات پیشرفته** با ۱۰۰ ذره در حال حرکت
- حرکت تصادفی در تمام جهات با تغییر جهت هنگام برخورد به لبه‌ها
- **بهینه‌شده برای نمایشگرهای Retina و 4K** با پشتیبانی از DPR (Device Pixel Ratio)

#### سرویس‌ها

- **جلوه پرسپکتیو سه‌بعدی** با استفاده از پکیج `react-parallax-tilt`
- حرکت عناصر با حرکت ماوس برای تجربه جذاب‌تر

### 📱 بخش‌های اصلی

#### فرم تماس (Contact Form)

- **اعتبارسنجی کامل فرم** (Form Validation)
- مدیریت خطاها و نمایش پیام‌های مناسب
- ارسال اطلاعات به سرور

#### اسلایدر

- **کروسل ریسپانسیو** برای نمایش نظرات مشتریان یا نمونه کارها
